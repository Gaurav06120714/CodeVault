# 💡 Patents

> Protecting genuinely novel inventions. Rarely worthwhile for a CRUD/sync SaaS like CodeVault, but documented for completeness.

| Field | Detail |
|-------|--------|
| **Overview** | Legal protection for novel, non-obvious inventions (utility patents). |
| **Purpose** | Exclusive rights to a novel technical method. |
| **Category** | 🏢 Enterprise Only (rare) |
| **Why it is needed** | Only if CodeVault invents something genuinely novel + patentable; usually not the case for sync/analytics. |
| **Legally required?** | No. |
| **Technically required?** | No. |
| **When to implement** | Only if a truly novel method emerges + there's strategic value. |
| **Priority** | ⚪ Low / unlikely |
| **Estimated Cost** | $8k–$30k+ per patent (attorney + filing) + maintenance. |
| **Renewal** | Maintenance fees (e.g. US: 3.5/7.5/11.5 yrs). |
| **Official Website** | https://www.uspto.gov |
| **Eligibility** | Novel, non-obvious, useful invention. |

## Step-by-Step Process
1. Identify a potentially novel method; do a prior-art search.
2. Consult a patent attorney on patentability/value.
3. File provisional → non-provisional if justified.

## Required Documents
- Invention disclosure; prior-art search; drawings/specs.

## Implementation Guide
- Most CodeVault functionality (OAuth, sync, stats aggregation) is well-known prior art → not patentable.
- Prefer trade-secret + copyright + trademark protection instead.

## Best Practices
- Don't patent for vanity; patents are costly + public; only pursue with clear strategic ROI.

## Common Mistakes
- Trying to patent obvious/known techniques; public disclosure before filing (bars patentability).

## CodeVault-specific Notes
- **Unlikely needed.** Protect IP via [COPYRIGHT](COPYRIGHT.md) + [TRADEMARK](TRADEMARK.md) + trade secrets instead.

## Future Considerations
- Revisit only if a novel algorithm/method is invented with defensible value.

## Checklist
- [ ] Novelty + prior-art assessed (likely none)
- [ ] Attorney consulted only if genuinely novel
- [ ] Otherwise rely on copyright/trademark/trade-secret
